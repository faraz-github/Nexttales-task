# Nexttales-task

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Commands

- npm install
- npm start

## Installation

Nexttales-task requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and start the server.

```sh
cd Nexttales-task
npm i
npm start
```
